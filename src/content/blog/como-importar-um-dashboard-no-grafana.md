## Como Importar um Dashboard no Grafana

Neste tutorial, você vai aprender a importar um dashboard no Grafana de forma simples e prática. Siga os passos abaixo:

## Acesse o Grafana

Você deve abrir o navegador e acessar o endereço do Grafana. Geralmente, ele está disponível em http://localhost:3000 (se estiver rodando localmente) ou em um endereço fornecido pela sua equipe.  
Faça login no Grafana com o seu usuário e senha.

## Navegue até a Seção de Dashboards

No menu lateral esquerdo, você deve clicar em "Dashboards" (ou "Painéis").  
Em seguida, clique em "Browse" (ou "Navegar") para ver os dashboards existentes.

Importe um Dashboard  
No canto superior direito, você deve clicar no botão "New" (ou "Novo") e selecionar "Import" (ou "Importar").  
Isso levará você para a página de importação.

![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_01-1024x159.png)

## Escolha o Método de Importação  
##   
Na página de importação, você verá três opções:

Upload JSON file: Use essa opção se você tiver um arquivo JSON do dashboard.

Import via panel JSON: Se você tiver o código JSON, cole-o na caixa de texto.

Import via grafana.com: Use essa opção para importar um dashboard público do repositório do Grafana.

![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_02-1024x703.png) ![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_03.png) ![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_04.png)

Escolha o Data Source (fonte de dados) que será usado. Se necessário, configure a fonte de dados antes.  
Clique em "Import" (ou "Importar") para finalizar.

## Visualize e Personalize o Dashboard  
##   
Após a importação, o dashboard será aberto automaticamente.  
Você pode explorar os painéis e gráficos para verificar se tudo está funcionando.  
Se precisar, faça ajustes nos painéis, como alterar métricas, títulos ou estilos.

![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_05-1024x374.png)

## Salve o Dashboard  
##   
Se você fez personalizações, não se esqueça de salvar o dashboard clicando no ícone de "Save" (ou "Salvar") no canto superior direito.  
Dê um nome ao dashboard e escolha uma pasta para organizá-lo.

![](https://blog.rosnertech.com.br/wp-content/uploads/2025/02/dash_06-1024x202.png)

## Dicas Extras  
##   
Dashboards Públicos: Você pode encontrar muitos dashboards prontos no grafana.com. Eles são ótimos para inspirar ou adaptar às suas necessidades.  
Exportação: Se você quiser exportar um dashboard no futuro, basta acessar o dashboard desejado, clicar no ícone de engrenagem (configurações) e selecionar "Export" (ou "Exportar").
